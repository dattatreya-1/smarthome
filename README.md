# SmartHome - Minimal Template

This repository contains a minimal Express + WebSocket + mock AI + mock Object Storage scaffold.

## Run Locally
1. Open terminal:
   ```
   cd server
   npm install
   npm run dev
   ```
   Server runs on http://localhost:5000

2. Open client/index.html in browser to view placeholder.

## Push to GitHub
```
git init
git add .
git commit -m "Add SmartHome template"
git branch -M main
git remote add origin https://github.com/dattatreya-1/smarthome.git
git push -u origin main --force
```
